## Problems encountered during development

So when I was building the header (list-header) component there was a problem with being able to press on the link for the shopping cart and so open the modal. It seemed that the pressable's view was preventing the link being selected!?! A quick search revealed the following [issue](https://github.com/expo/expo/issues/23619) in the expo project. It wasn't the initial query, rather a follow up where the guy said he was having trouble with styling and a response was to use asChild "as described in the [expo docs](https://docs.expo.dev/router/basics/navigation/#buttons)". Remarkably doing this did fix the problem!
